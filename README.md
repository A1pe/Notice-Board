# 저의 게시판 만들기 프로젝트에 온 것을 환영합니다!

## Description
  해당 프로젝트는 제가 여태까지 배운 것들을 활용하여 간단한 게시판을 만드는 것에 목적을 두고 있으며
  
  글작성, 글수정, 댓글 서비스를 제공합니다.
  
  해당 README는 프로젝트를 진행하며 생기는 일일 목표와 목표 달성량을 기록하는 것에 중점을 두고 있습니다.
